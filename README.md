📊 Searamart Talent Analytics Dashboard

An end-to-end Data Analytics portfolio project that analyzes candidate demographics, technical skills, recruitment platforms, and career goals within a simulated Retail & E-Commerce Talent Pool.

---

## 🚀 Project Overview

This project demonstrates a complete Data Analytics workflow:

- SQL for data exploration
- Python (Pandas) for data cleaning
- Power BI for dashboard visualization
- Business insights through interactive reporting

## 📋 Executive Summary

- 🛠 SQL is the most common technical skill, possessed by 561 candidates.
- 🎯 Nearly half of the candidates (48.75%) are classified as Advanced professionals.
- 📍 West Java contributes the largest talent pool with 173 registered candidates.
- 🌐 LinkedIn is the primary recruitment platform, accounting for 549 candidates.
